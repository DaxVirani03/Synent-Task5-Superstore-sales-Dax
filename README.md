# Dax Virani - Superstore Analysis

Objective
- Perform BI-like sales analysis to surface revenue, profit, and category-level insights from the Superstore dataset.

Steps performed
- Load Superstore CSV and clean data types
- Aggregate sales and profit by time and category
- Create visualizations for trends, top products, and regional performance

Tools / Libraries
- pandas, numpy, matplotlib, seaborn

Outcome (brief)
- Notebook delivers summary dashboards and charts useful for operational and strategic decisions.

How to run

```bash
pip install -r requirements.txt
jupyter lab
jupyter nbconvert --to notebook --execute "Dax Virani - superstore.ipynb" --inplace
```

Notes
- Place the Superstore CSV in the workspace; notebook uses relative paths.
